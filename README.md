### Running Haxe  
* Interpeter: 
    `haxe -main Day03 --interp`
* Neko: 
    `haxe -main Day03 -neko day03.n`
* CPP Linux/Mac: 
    `haxe -main Day03 -cpp bin`
* CPP Windows: 
    `haxelib install minimingw` 
    `haxe -main Day03 -cpp bin -lib minimingw`
