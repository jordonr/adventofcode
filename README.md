### Running Haxe  
* Interpeter: 
    `haxe -main Day01 --interp`
* Neko: 
    `haxe -main Day01 -neko day01.n`
* CPP Linux/Mac: 
    `haxe -main Day01 -cpp bin`
* CPP Windows: 
    `haxelib install minimingw` 
    `haxe -main Day01 -cpp bin -lib minimingw`
